# Programa-advogacia

## Tecnologias Utilizadas
| Biblioteca / Ferramenta | Função / Descrição  |
|-------------------------|------------------------------------|
| Dash                    | Framework web principal |
| Dash Bootstrap          | Componentes UI estilizados |
| SQLite3                  | Armazenamento de usuários |
| Hashlib                 | Criptografia de senhas  |
| Python Puro             | Lógica de negócio   |
| Session Storage         | Gerenciamento de estado do usuário |


##  Como utilizar

### 1. Clone o repositório
```bash
git clone https://github.com/gleiSUN/Programa-advocacia.git
```

### 2. Acesse a pasta do projeto
```bash
cd Programa-advocacia
```

### 3. Crie um ambiente virtual (venv)
```bash
python -m venv venv
```

### 4. Ative o ambiente virtual
### Windows:
```bash
venv\Scripts\activate
```

### Linux/macOS:
```bash
source venv/bin/activate
```
### 5. Instale as dependências
```bash
pip install -r requirements.txt
```

### 6. Execute o projeto
```bash
python index.py
```
